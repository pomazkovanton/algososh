# Проектная работа. МБОУ АЛГОСОШ им. Фибоначчи

Данный проект является визуализатором популярных алгоритмов. Проект заточен на анимацию и поэтапное отображение работы алгоритма, что позволит детальнее понять каждый шаг его работы.
